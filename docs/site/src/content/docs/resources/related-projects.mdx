This integration was inspired by [astro-selfie](https://github.com/vadimdemedes/astro-selfie) and a similar plugin I wrote for [Gatsby](https://www.gatsbyjs.com/) called
[Gatsby Plugin: Component to Image](https://github.com/twocaretcat/gatsby-plugin-component-to-image).

Compared to astro-selfie, this integration exposes a lot more configuration options which allow you to completely customize how images are generated.
